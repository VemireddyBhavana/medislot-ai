**Built for Theme 6: Booking, Scheduling & Workforce**

## ?? The Problem
Clinics often manage appointments manually, leading to double bookings, unoptimized doctor workloads, high no-show rates, and overall inefficient appointment handling. Patients suffer from long wait times, and clinic staff burn out from unevenly distributed workloads.

## ?? The Solution: MediSlot AI
MediSlot AI is a next-generation healthcare SaaS that eliminates manual scheduling inefficiencies through intelligent, automated workflows.

### ?? Core Smart Features
- **Smart Slot Recommendations**: The booking engine dynamically suggests optimal appointment times to balance the daily load across available doctors.
- **No-Show Prediction**: Appointments are automatically tagged with a low/medium/high no-show risk based on simulated historic patterns, allowing staff to proactively confirm high-risk visits.
- **Workload Balancer**: The admin dashboard proactively alerts staff if a doctor is overloaded and suggests re-routing routine visits to under-utilized staff.
- **Automated Reminders**: Built-in automated notification logging for confirmations, reminders, and follow-ups.
- **Dynamic Triage**: Urgent appointments are visually highlighted on the live control board for immediate staff attention.

## ?? Tech Stack
- **Frontend**: React (Vite) + Tailwind CSS (v4) + Lucide Icons
- **Backend**: Node.js + Express.js
- **Database**: MongoDB (Mongoose)

## ?? Demo Script Flow
1. **Patient Booking**: Open http://localhost:5173. Click "Book Appointment". Show the "Smart Slots" and "Workload Hint" on the right panel. Book an "Urgent" appointment.
2. **Confirmation**: Show the polished confirmation receipt.
3. **Admin Dashboard**: Go to http://localhost:5173/admin/login and login. Show the live metrics and the AI insights (Overloaded Doctor alert).
4. **Live Control Board**: Navigate to "Manage Appointments". Point out how the "Urgent" appointment you just booked is automatically tinted red, and "High Risk" no-shows are tinted orange. Instantly update a status from "Booked" to "Complete" using the inline dropdown.
5. **System Notifications**: Navigate to "Notifications" to show the automated message queue generated by the smart booking engine.
